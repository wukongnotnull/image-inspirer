## 使用演示

### 场景一：搜索成功案例，直接复用

已经有明确需求，直接搜案例参考：

```
用户      ❯ 有没有信息图可视化的案例？

image-inspirer  ❯ 读取 图表与信息可视化/prompt.md，找到 55 个信息图案例

                  【案例 1】城市生命系统图谱
                  类型：信息图可视化 | 比例：9:16

                  城市生命系统图谱 / Urban Metabolism Atlas
                  Vertical 9:16 isometric cutaway infographic "城市生命系统图谱"。
                  Smart city from sky to bedrock: skyscrapers, streets, subway...
                  Style: engineering white paper + scientific atlas, 8K.

                  【案例 2】AI 神经网络原理图
                  ...
```

### 场景二：模糊需求，启发引导

不知道要什么，帮你理清方向：

```
用户      ❯ 给我一个好看的图片

image-inspirer  ❯ 启动启发框架，逐一提问：

                  Q1: 你想要什么类型？
                  → 海报 / 摄影 / 插画 / 界面 / 其他

                  Q2: 用途是什么？
                  → 社媒传播 / 电商 / 个人收藏 / 商业提案

                  Q3: 风格偏好？
                  → 写实 / 电影感 / 极简 / 插画 / 国风

                  根据回答提供 2-3 个方向，确认后进入创作流程。
```

### 场景三：定制创作，模板+案例

有明确需求但需要定制化提示词：

```
用户      ❯ 为"AI之道"科技公司设计一张品牌海报

image-inspirer  ❯ 读取 海报与排版/prompt.md 品牌海报案例
                  ↓ 二次创作

                  生成一张高端 AI 科技公司品牌海报，竖版 9:16 比例。
                  Logo：八卦太极与神经网络融合图形，纯黑线条 + 青色光晕
                  主标题："AI之道"（中文宋体，极细字重）
                  副标题："Dao of Artificial Intelligence"
                  配色：纯白背景 + 纯黑主色 + 青色点缀
                  风格：极简主义，大量留白，高级感
                  约束：文字必须清晰可读，不要霓虹灯效果
```

### 场景四：运动品牌海报

```
用户      ❯ 帮我生成一个运动品牌的商业海报，要力量感

image-inspirer  ❯ 读取 海报与排版/prompt.md 运动海报案例

                  设计一张运动品牌商业 Campaign 海报。
                  主体：运动员/肌肉男，姿态：力量动作。
                  核心道具：哑铃/拳套，以夸张比例成为视觉锚点。
                  大字标题："突破极限"
                  视觉风格：高端运动品牌广告，强光影，反光地面。
                  约束：主体清晰，文字可读，色调统一。
                  输出：4:5，适合社媒传播的运动商业视觉。
```

## 避坑指南

### 通用原则

| 原则 | 说明 |
|------|------|
| 类型明确 | 海报/UI/摄影等类型要清晰定义 |
| 主体具体 | 描述具体的人/物/场景，不要模糊 |
| 风格锁定 | 插画风/摄影风/电影感要明确指定 |
| 参数完整 | 比例、分辨率、构图等关键参数要包含 |
| 文字可读 | 如需显示文字，必须明确要求文字清晰可读 |

### 各类型重点

| 类型 | 重点 | 常见翻车点 |
|------|------|-----------|
| UI界面 | 明确平台 + 比例 + 布局 | 模糊指令导致布局混乱 |
| 海报设计 | 主视觉具体，主副标题写死 | 文字模糊、排版错乱 |
| 摄影 | 加入皮肤纹理、胶片颗粒 | AI 塑料感、过度光滑 |
| 插画 | 锁定笔触风格 | 默认塑料风、风格漂移 |
| 电商 | 产品突出 + 光影真实 | 产品变形、光影不统一 |
| Logo | 可识别性 + 可延展性 | 过于复杂、不可缩放 |
