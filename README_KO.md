## 데모

### 시나리오 1: 검색 및 재사용

필요한 것을 이미 알고 있다면, 직접 사례 검색:

```
사용자      ❯ 인포그래픽 시각화 사례가 있나요?

image-inspirer  ❯ 图表与信息可视化/prompt.md 읽는 중, 55개의 인포그래픽 사례 발견

                  【사례 1】Urban Metabolism Atlas
                  유형: 인포그래픽 | 비율: 9:16

                  城市生命系统图谱 / Urban Metabolism Atlas
                  Vertical 9:16 isometric cutaway infographic "城市生命系统图谱"。
                  Smart city from sky to bedrock: skyscrapers, streets, subway...
                  Style: engineering white paper + scientific atlas, 8K.

                  【사례 2】AI 신경망 다이어그램
                  ...
```

### 시나리오 2: 모호한 요구, 가이드형 발견

무엇을 원하는지 모르시나요? 방향을 찾도록 도와드립니다:

```
사용자      ❯ 멋진 이미지를 주세요

image-inspirer  ❯ 영감 프레임워크 시작, 순차적으로 질문:

                  Q1: 어떤 유형을 원하시나요?
                  → 포스터 / 사진 / 일러스트 / UI / 기타

                  Q2: 용도는 무엇인가요?
                  → SNS / 이커머스 / 개인 컬렉션 / 비즈니스 제안

                  Q3: 스타일 선호도는?
                  → 사실적 / 시네마틱 / 미니멀 / 일러스트 / 중국풍

                  답변에 따라 2~3가지 방향을 제안합니다.
```

### 시나리오 3: 맞춤 제작, 템플릿 + 사례

명확한 요구사항이 있지만 맞춤형 프롬프트가 필요할 때:

```
사용자      ❯ "AI之道" 기술 회사의 브랜드 포스터를 디자인해주세요

image-inspirer  ❯ 海报与排版/prompt.md 브랜드 포스터 사례 읽는 중
                  ↓ 맞춤 제작

                  프리미엄 AI 기술 회사 브랜드 포스터, 세로형 9:16 비율.
                  로고: 팔괘 태극과 신경망 융합, 순수 검은 선 + 시안 글로우
                  타이틀: "AI之道" (송체, 극세 굵기)
                  서브타이틀: "Dao of Artificial Intelligence"
                  색상: 순백 배경 + 순흑 + 시안 포인트
                  스타일: 미니멀리스트, 넉넉한 여백, 프리미엄 감성
                  제약: 텍스트는 명확히 읽을 수 있어야 함, 네온 효과 없음
```

### 시나리오 4: 스포츠 브랜드 포스터

```
사용자      ❯ 힘이 느껴지는 스포츠 브랜드 상업 포스터를 생성해주세요

image-inspirer  ❯ 海报与排版/prompt.md 스포츠 포스터 사례 읽는 중

                  스포츠 브랜드 상업 Campaign 포스터 디자인.
                  피사체: 운동선수/근육질 남성, 파워 포즈.
                  주요 소품: 덤벨/권투 글러브, 과장된 비율로 시각적 앵커.
                  헤드라인: "한계를 돌파하라"
                  비주얼 스타일: 프리미엄 스포츠 브랜드 광고, 극적인 조명, 반사 바닥.
                  제약: 피사체 선명, 텍스트 가독성, 통일된 톤.
                  출력: 4:5, SNS 최적화된 스포츠 상업 비주얼.
```

## 설치

### 방법 1: CLI (개발자용)

npx로 직접 설치:

```bash
npx skills add wukongnotnull/image-inspirer
```

설치 후, Agent에게 말하세요:

```markdown
> 인포그래픽 시각화 사례 찾아줘
> 중국풍 포스터 프롬프트 생성해줘
> 멋진 이미지 하나 줘
```

### 방법 2: 대화형 (비개발자용)

이것을 Agent에게 복사하세요:

```bash
이 스킬 설치: https://github.com/wukongnotnull/image-inspirer
```

## 함정 가이드

### 일반 원칙

| 원칙 | 설명 |
|------|------|
| 명확한 유형 | 포스터/UI/사진 등을 명확히 정의 |
| 구체적인 피사체 | 특정 인물/사물/장면을 설명, 모호함 방지 |
| 고정된 스타일 | 일러스트/사진/시네마틱을 명시적으로 지정 |
| 완전한 매개변수 | 비율, 해상도, 구도 등 주요 매개변수 포함 |
| 읽기 쉬운 텍스트 | 텍스트가 필요하면 명확히 읽을 수 있도록 요구 |

### 유형별 핵심 사항

| 유형 | 중점 | 흔한 실수 |
|------|------|----------|
| UI | 명확한 플랫폼 + 비율 + 레이아웃 | 모호한 지시로 레이아웃 혼란 |
| 포스터 | 구체적인 메인 비주얼, 제목/부제목 고정 | 텍스트 흐림, 레이아웃 오류 |
| 사진 | 피부 질감, 필름 그레인 추가 | AI 플라스틱 룩, 과도하게 매끄러움 |
| 일러스트 | 브러시 스타일 고정 | 기본 플라스틱 룩, 스타일 드리프트 |
| 이커머스 | 제품 강조 + 사실적인 조명 | 제품 왜곡, 조명 불일치 |
| 로고 | 인식성 + 확장성 | 너무 복잡함, 확장 불가 |
