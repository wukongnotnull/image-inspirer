## デモ

### シナリオ1：検索して再利用

必要なものが明確な場合、直接事例を検索：

```
ユーザー    ❯ インフォグラフィックの事例はありますか？

image-inspirer  ❯ 图表与信息可视化/prompt.md を読み込み、55件のインフォグラフィック事例を発見

                  【事例1】Urban Metabolism Atlas
                  タイプ：インフォグラフィック | 比率：9:16

                  城市生命系统图谱 / Urban Metabolism Atlas
                  Vertical 9:16 isometric cutaway infographic "城市生命系统图谱"。
                  Smart city from sky to bedrock: skyscrapers, streets, subway...
                  Style: engineering white paper + scientific atlas, 8K.

                  【事例2】AIニューラルネットワーク図
                  ...
```

### シナリオ2：あいまいな要件、ガイド付き発見

何が欲しいかわからない？方向性を見つけるお手伝いをします：

```
ユーザー    ❯ いい感じの画像をください

image-inspirer  ❯ インスピレーションフレームワークを起動、順に質問：

                  Q1: どのタイプが欲しいですか？
                  → ポスター / 写真 / イラスト / UI / その他

                  Q2: 用途は？
                  → SNS / EC / 個人コレクション / ビジネス提案

                  Q3: スタイルの好みは？
                  → フォトリアル / シネマティック / ミニマル / イラスト / 中華風

                  回答に基づいて2〜3の方向性を提案します。
```

### シナリオ3：カスタム作成、テンプレート＋事例

要件は明確だがカスタマイズされたプロンプトが必要：

```
ユーザー    ❯ "AI之道"テック企業のブランドポスターをデザインして

image-inspirer  ❯ 海报与排版/prompt.md のブランドポスター事例を読み込み
                  ↓ カスタム作成

                  高級AIテック企業ブランドポスター、縦型9:16比率。
                  ロゴ：八卦太極とニューラルネットワークの融合、純黒線＋シアンの輝き
                  タイトル：「AI之道」（宋体、極細ウェイト）
                  サブタイトル：「Dao of Artificial Intelligence」
                  配色：純白背景＋純黒＋シアンアクセント
                  スタイル：ミニマリスト、十分な余白、高級感
                  制約：テキストは明確に読みやすく、ネオン効果なし
```

### シナリオ4：スポーツブランドポスター

```
ユーザー    ❯ 力強さを感じるスポーツブランドの商業ポスターを生成して

image-inspirer  ❯ 海报与排版/prompt.md のスポーツポスター事例を読み込み
```

## インストール

### 方法1：CLI（開発者向け）

npxで直接インストール：

```bash
npx skills add wukongnotnull/image-inspirer
```

インストール後、Agentに伝えてください：

```markdown
> インフォグラフィックの事例を探して
> 中華風ポスターのプロンプトを生成して
> いい感じの画像をください
```

### 方法2：会話型（非開発者向け）

これをAgentにコピーしてください：

```bash
このスキルをインストール：https://github.com/wukongnotnull/image-inspirer
```

## 落とし穴ガイド

### 一般原則

| 原則 | 説明 |
|------|------|
| 明確なタイプ | ポスター/UI/写真などを明確に定義 |
| 具体的な被写体 | 特定の人/物/シーンを記述、曖昧さを避ける |
| 固定されたスタイル | イラスト/写真/シネマティックを明示的に指定 |
| 完全なパラメータ | 比率、解像度、構図などの重要パラメータを含める |
| 読みやすいテキスト | テキストが必要な場合、明確に読みやすさを要求 |

### タイプ別のポイント

| タイプ | 重点 | よくある失敗 |
|--------|------|-------------|
| UI | 明確なプラットフォーム＋比率＋レイアウト | 曖昧な指示でレイアウトが混乱 |
| ポスター | 具体的なメインビジュアル、タイトル/サブタイトルを固定 | テキストがぼやける、レイアウトエラー |
| 写真 | 肌の質感、フィルムグレインを追加 | AIプラスチック感、過度に滑らか |
| イラスト | 筆致スタイルを固定 | デフォルトのプラスチック感、スタイルのブレ |
| Eコマース | 製品の強調＋リアルな照明 | 製品の変形、照明の不一致 |
| ロゴ | 認識性＋拡張性 | 複雑すぎる、スケーラブルでない |
